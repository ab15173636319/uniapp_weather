# 天气预报应用

一个基于 Uniapp 开发的美观、功能完善的天气预报应用。

<img src="menu.png" />

## 功能特点

- **实时天气查询**：自动获取当前位置的天气信息
- **温度信息**：显示最高温度和最低温度
- **天气预报**：展示未来几天的天气状况和温度范围
- **生活指数**：提供空调、洗车、穿着、感冒、体感、晒床单、锻炼、紫外线等生活指数建议
- **美观界面**：采用渐变色彩、半透明效果和微妙动画，打造唯美视觉体验
- **响应式设计**：适配不同屏幕尺寸

## 核心功能模块

### 1. 天气查询

应用通过调用外部 API 获取当前位置的天气信息，包括实时天气、温度范围、天气预报和生活指数。

### 2. 生活指数

提供详细的生活指数信息，每个指数包含等级、简要描述和详细建议：

- **空调指数**：建议是否开启空调及模式
- **洗车指数**：天气是否适合洗车
- **穿着指数**：建议穿着的衣物类型
- **感冒指数**：感冒风险等级及预防建议
- **体感指数**：体感温度描述
- **晒床单指数**：是否适合晾晒衣物
- **锻炼指数**：是否适合户外运动
- **紫外线指数**：紫外线强度及防护建议

### 3. 视觉设计

- **渐变背景**：采用紫色到粉色到橙色的渐变色彩，营造唯美氛围
- **半透明效果**：使用半透明背景和边框，增强现代感
- **动画效果**：添加光斑脉冲、卡片悬停和滑动动画
- **响应式布局**：适配不同屏幕尺寸，确保在各种设备上都有良好的显示效果

## 技术实现

- **框架**：Uniapp
- **前端语言**：Vue 3 + JavaScript
- **样式**：SCSS
- **网络请求**：自定义 http 工具
- **天气 API**：使用 uapis.cn 提供的天气接口

## 安装使用

1. **克隆项目**

2. **安装依赖**
   ```bash
   npm install
   ```

3. **运行项目**
   - 开发环境：`npm run dev:h5`
   - 构建生产版本：`npm run build:h5`

4. **预览**
   - H5 端：直接在浏览器中打开生成的 index.html 文件
   - 小程序端：使用对应的小程序开发工具导入项目

## 注意事项

- 应用依赖于外部天气 API，需要确保网络连接正常
- 首次加载时可能需要获取位置权限，请根据提示授权
- 部分动画效果可能在低端设备上性能受限
